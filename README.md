Portfolio Description – IT 116: Multimedia Systems

my_portfolio_web is a personal portfolio website showcasing all my major activities and creative outputs from IT 116 – Multimedia Systems. This project highlights the multimedia skills I developed throughout the semester, presented through the following key features:

Image Editing (Photoshop or Canva) – Edited images demonstrating enhancement, manipulation, and creative design techniques.

Brand Logo with Brand Colors – A custom-designed brand logo along with an organized color palette representing my chosen brand identity.

Photoshop Editing Activity – A collection of Photoshop-based tasks showcasing advanced image correction, retouching, layout composition, and visual creativity.

Audio Editing – Audio files and editing outputs demonstrating cutting, mixing, enhancement, and sound design skills.

This portfolio brings together all my multimedia work in one organized platform, highlighting my growth in digital design, creativity, and media production.
